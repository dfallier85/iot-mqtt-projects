# IoT MQTT Projects

## Overview
This repository contains MQTT-based communication projects for robotics, embedded devices, dashboards, and broker-driven system integration.

## Topics Covered
- MQTT topic design
- Broker setup on Raspberry Pi
- Node-RED to MQTT pipelines
- Robot command and response flows
- Portable IoT hub development
- Dashboard-based device interaction

This repo is part of my robotics and automation portfolio and demonstrates how connected devices and robots can communicate reliably through lightweight messaging workflows.
